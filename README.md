#  Supply Chain Analysis Project

Project Overview
This project analyzes supply chain data to identify trends, improve operational efficiency, 
and support business decision-making. The analysis focuses on inventory management, 
sales performance, supplier performance, and delivery efficiency.

 Objectives
- Analyze product sales trends
- Monitor inventory levels
- Evaluate supplier performance
- Track delivery and shipping efficiency
- Identify areas for cost optimization

 Tools & Technologies
- Power BI
- SQL
- Excel

Key Insights
- Identified top-performing products
- Analyzed inventory turnover
- Evaluated supplier reliability
- Monitored delivery performance
- Highlighted opportunities for reducing operational costs

Dashboard Features
- Sales Overview
- Inventory Analysis
- Supplier Performance
- Shipping & Delivery Tracking
- KPI Metrics

Project Files
- Supply_Chain_Dashboard.pbix
- Supply_Chain_Dataset.xlsx
- Dashboard_Screenshots

How to Use
1. Download the project files.
2. Open the `.pbix` file in Power BI Desktop.
3. Refresh the dataset if needed.
4. Explore the interactive dashboard.
